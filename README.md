# Bank of React

This repository contains my implementation of the Bank of React project for CSCI 395 Assignment 3. It uses React Router to simulate a simple banking application with pages for home, login, user profile, credits, and debits.

## Project Description
This application demonstrates client-side routing and state management in React. It simulates a personal banking interface and includes the following functionality:

- Display account balance
- View credits and debits
- Add new credits and debits
- Update balance based on user input and API values
- Login functionality with mock state update
- Navigation through React Router (v5)

## Technical Features
- React class components
- BrowserRouter, Route, Link, and Redirect from React Router
- API data retrieval for credits and debits
- State passed as props to child components
- Dynamic balance calculation
